# 🍽️ نظام الإدارة المتكامل للمطاعم والكافيهات

نظام برمجي متكامل ومستقل لإدارة المطاعم، مبني للعمل محلياً عبر متصفح الويب لضمان السرعة، الخصوصية، والمرونة العالية. يغطي النظام كافة العمليات اليومية بدءاً من شاشات الكاشير وإدارة الطاولات، وصولاً إلى تقارير المخازن المعقدة لحركة الخامات.

---

## ✨ المميزات الرئيسية للمنظومة

### 🛒 المبيعات وإدارة الطلبات
* **أنظمة طلبات متعددة:** دعم كامل لطلبات الصالة (Dine-in)، التيك أواي (Takeaway)، والدليفري (Delivery).
* **إدارة مرنة للتوصيل:** إمكانية تحديد تسعيرة الدليفري ديناميكياً بناءً على المنطقة.
* **تتبع الطلبات:** نظام متكامل لمتابعة حالة الطلبات خطوة بخطوة مع إمكانية الإلغاء عند الحاجة.
* **العروض والوجبات:** وحدة خاصة لإنشاء وتخصيص الوجبات المجمعة والعروض الخاصة.

### 💳 المدفوعات والماليات
* **طرق دفع متعددة:** دعم الدفع النقدي (Cash)، البطاقات الائتمانية (Visa)، وطرق الدفع الأخرى بحسب الحاجة.
* **تقارير مالية شاملة:** استخراج تقارير يومية، شهرية، أو مخصصة حسب فترة زمنية معينة لتقييم الأداء بدقة.

### 📦 إدارة المخازن والخامات (Inventory)
* **دورة مخزنية كاملة:** إدارة دقيقة للمواد الخام (Raw Materials).
* **حركة الأصناف:** تسجيل حركة الوارد والصادر ومتابعة الأرصدة الحالية لحظة بلحظة لمنع الهدر.

### ⚙️ الإدارة والتشغيل
* **هيكلة المنيو:** تقسيم المنيو إلى مجموعات (Categories) لسهولة التصفح.
* **واجهة بصرية:** جميع الأصناف مدعومة بالصور لضمان سرعة الاستخدام وتقليل الأخطاء من قبل الكاشير.
* **إدارة طاولات الصالة:** شاشة مخصصة لمتابعة حالة الطاولات المتاحة والمشغولة.
* **نظام الصلاحيات:** لوحة تحكم لتحديد صلاحيات المستخدمين (كاشير، مدير، مخازن) لضمان أمان النظام.

### 💻 المواصفات التقنية
* **يعمل على أي جهاز:** بفضل تصميمه كتطبيق ويب محلي (Local Web App)، يمكن تشغيله على أجهزة الكمبيوتر، الأجهزة اللوحية، والهواتف الذكية عبر الشبكة الداخلية.
* **أمان البيانات:** يوفر النظام آلية سلسة جداً لعمل النسخ الاحتياطي (Backup) مع إمكانية استرجاع البيانات (Restore) بضغطة زر.
